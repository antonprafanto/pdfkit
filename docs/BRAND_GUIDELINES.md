# PDF Kit Brand Guidelines

## Brand Identity

### Application Name
**PDF Kit** - A comprehensive, offline-first PDF toolkit.

### Tagline
"Your Complete PDF Toolkit - Privacy First, Always Offline"

### Brand Values
- **Privacy**: Documents never leave your device
- **Simplicity**: Easy-to-use interface for everyone
- **Completeness**: All PDF tools in one app
- **Open Source**: Free and transparent

---

## Logo & Icon

### Primary Icon
The PDF Kit icon consists of:
1. **Blue gradient circle** - Represents trust and reliability
2. **White document** - Represents the PDF document
3. **"PDF" text** - Clear identification
4. **"Kit" badge** - Distinguishes as a complete toolkit

### Color Variants
- **Full color**: Blue gradient with white document
- **Light mode**: Blue icon on white background
- **Dark mode**: Blue icon on dark background
- **Monochrome**: Single color for printing

### Icon Sizes
| Platform | Size | Format |
|----------|------|--------|
| Windows | 256x256 | .ico |
| macOS | 512x512 | .icns |
| Linux | 512x512 | .png |
| Favicon | 32x32, 16x16 | .ico |

## Typography

### Primary Font
**Inter** - Clean, modern, and highly readable

### Font Weights
- **Regular (400)**: Body text
- **Medium (500)**: Labels, buttons
- **Semibold (600)**: Headings, emphasis
- **Bold (700)**: Primary headings

## UI Components

### Buttons
- **Primary**: Blue background, white text
- **Secondary**: White background, blue border
- **Ghost**: Transparent, blue text
- **Danger**: Red background, white text

### Border Radius
- **Small**: 4px (inputs, small buttons)
- **Medium**: 6px (buttons, cards)
- **Large**: 12px (modals, large cards)
- **Full**: 9999px (pills, avatars)

### Shadows
```css
--shadow-sm: 0 1px 2px rgba(0, 0, 0, 0.05);
--shadow-md: 0 4px 6px rgba(0, 0, 0, 0.1);
--shadow-lg: 0 10px 15px rgba(0, 0, 0, 0.1);
--shadow-xl: 0 20px 25px rgba(0, 0, 0, 0.15);
```
