## ğŸ”§ Essential Operations

### 1. Split a PDF

Need to extract specific pages?

1. Open your PDF
2. Click **Split** in the toolbar
3. Enter page range: `1-5` or `1, 3, 5-10`
4. Click **Split**
5. Choose where to save

### 2. Merge PDFs

Combine multiple PDFs into one:

1. Click **Merge** in the toolbar
2. Click **Add Files** to select PDFs
3. Drag to reorder if needed
4. Click **Merge**
5. Save your combined PDF

### 3. Compress PDF

Reduce file size for sharing:

1. Open your PDF
2. Click **Compress** in the toolbar
3. Select quality level
4. Click **Compress**
5. Save the smaller file

### 4. Add Password

Protect sensitive documents:

1. Open your PDF
2. Go to Security â†’ Encrypt
3. Enter a strong password
4. Click **Encrypt**
5. Save the protected PDF
